## ESTILO VISUAL BASE (incluir en todos los prompts de ilustraciones)

```
Estilo visual global de RelaxMind:
- Ilustraciones: flat design con formas orgánicas suaves, líneas redondeadas, sombras suaves
- Paleta: verde principal #0F6E56, verde claro #68D391, índigo #4338A8, coral #E8582A, fondo blanco y cremas suaves
- Sin texturas rugosas ni realismo fotográfico — estilo moderno tipo Headspace/Calm app
- Personas: diversas, representadas de forma abstracta/minimalista (siluetas o formas simples)
- Ambiente: tranquilo, reconfortante, hopeful — nunca oscuro ni ansioso
- Formato: PNG con fondo transparente, a menos que se indique lo contrario
```

## TIPS PARA MEJORES RESULTADOS

- En **Midjourney**: añadir `--ar 1:1 --style raw --q 2` para íconos de logros
- En **DALL·E 3**: ser muy literal con los colores HEX en el prompt ("bright emerald green #0F6E56")
- En **Leonardo AI**: usar el modelo "Kino XL" para ilustraciones y "3D Animation Style" para los clay
- Para los **clay icons**: buscar en Behance "clay icon set" para referencia visual y adjuntarla como imagen de estilo
- Los **avatares**: se pueden generar todos de una vez en Midjourney con `--tile` y luego recortar
- Las **ilustraciones de pantallas**: son referencias de diseño, no assets finales; usarlas en Figma como guía
- Para **LottieFiles**: filtrar por "free" y "After Effects source available" para poder editar colores
